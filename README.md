# Missioné par

Ce software suit les besoins des utilisateurs peuplant le service de la COMM.

## Objectifs

Ces utilisateurs génèrent un volume conséquent de fichiers dans plusieurs formats différents. L'objectif est d'automatiser les tâches répétitives liées à cette génération.

## Fonctionnalités principales

### Sélection et duplication de fichiers

- Sélectionner un fichier depuis l'interface
- Dupliquer le fichier vers plusieurs destinations
- Convertir le fichier dans différents formats d'extension à chaque destination

### Configuration des chemins

- Définir des chemins de destination par défaut
- Éviter la ressaisie manuelle des chemins à chaque utilisation
- Réduire au maximum les étapes du workflow

### Workflow simplifié

1. Sélectionner un fichier
2. Indiquer les options et chemins de duplication
3. Spécifier les extensions pour chaque destination
4. Exécuter l'action automatisée
